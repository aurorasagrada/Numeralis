# Aurora Sagrada - Sistema Numerológico Completo

## 🌟 Descrição
Sistema completo de numerologia com design whimsigoth vintage, incluindo análises pitagóricas, cabalísticas e compatibilidade amorosa.

## 📁 Estrutura do Projeto

### Arquivos Principais
- `aurora_sagrada_sistema_completo.html` - Sistema principal com todas as funcionalidades
- `RELATORIO_CORRECOES_AURORA_SAGRADA.md` - Relatório detalhado das correções realizadas
- `todo.md` - Lista de tarefas e progresso do desenvolvimento

### Cartas de Tarô
- `cartas_tarot/` - 81 arquivos HTML com interpretações completas das cartas
- `imagens_cartas/` - 158 imagens das cartas de tarô em alta qualidade

## ✨ Funcionalidades

### 🔮 Numerologia Pitagórica
- Número da Alma (motivação interna)
- Número da Personalidade (impressão externa)
- Número da Expressão (talentos e habilidades)
- Número do Destino (propósito de vida)
- Ano Pessoal (ciclo atual)

### 🔺 Numerologia Cabalística
- Pirâmide invertida numerológica
- Cálculo do Arcano Regente
- Determinação da posição atual na vida
- Interpretações baseadas nos 78 arcanos

### 💕 Compatibilidade Amorosa
- Análise completa entre duas pessoas
- Porcentagens por áreas (amor, paixão, espiritualidade, material)
- Identificação de forças e desafios do relacionamento
